Problème : La pos d'Alien est mal set dans le Main, il y a deux fois un set sur la valeur x.
Cause : Erreur d'inattention.
Solution : Juste remplacer la duplication a.pos.x par a.pos.y

Problème : Le DecreaseHealth() n'était pas virtual dans GameObject.
Cause : Oublie de virtual devant la fonction.
Solution : Rendre cette fonction virtual pour qu'elle puisse correctement être hérité.

Problème : (Cette erreur était drôle) les strings des noms PlayerShip et AlienLaser dans PlayField.
Cause : Erreur d'inattention.
Solution : Il suffit d'avoir une bonne vue et de mettre une majuscules à chaque nom.

Problème : Dans le Playfield, le controllerInput est tout simplement nullptr.
Cause : Il n'y a pas eu d'initialisation du PlayField, des bounds et des Inputs.
Solution : Il suffit de changer le constructeur pour initialiser tout ça.

Problème : AlienLaser, il n'y a pas de vérification du player, pour savoir s'il est nullptr ou non.
Cause : Oublie.
Solution : Il faut juste vérifier si le player est nullptr.

Problème : PlayerLaser, il y a un delete this dans l'update (Pas mal aussi celle-là).
Cause : Juste une envie de nuire je suppose.
Solution : Il suffit d'enlever le delete this pour ne plus l'avoir.

Problème : Il n'y a pas de destructor virtual pour supprimer les objets qui héritent de la class.
Cause : Cela n'a pas été fait.
Solution : Il suffit d'ajouter un destructor virtual.
